# typebeam

## Usage
in html, add code block:
```html
<script src='https://cdn.provider.com/typebeam.js?url=https://fonts.gstatic.com/s/geist/v4/gyByhwUxId8gMEwYGFWfOw.woff2'> // geist
```

## Params.
- `url`: the url of a remote font family file, support native html-supported types (`ttf`,`otf`,`etf`, `woff`, `woff2`,...)
