# typebeam

## Usage
Simply add code block:
```html
<script src='https://raw.githubusercontent.com/xolyn/typebeam/refs/heads/main/typebeam.js?url=https://fonts.gstatic.com/s/geist/v4/gyByhwUxId8gMEwYGFWfOw.woff2'> // example font: geist
```

### Speed up
If you are in China mainland, use:
```html
<script src='https://fastly.jsdelivr.net/gh/xolyn/typebeam/refs/heads/main/typebeam.js?url=https://fonts.gstatic.com/s/geist/v4/gyByhwUxId8gMEwYGFWfOw.woff2'> // example font: geist
```

## Params.
- `url`: the url of a remote font family file, support native html-supported types (`ttf`,`otf`,`etf`, `woff`, `woff2`,...)
